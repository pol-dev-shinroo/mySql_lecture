# command-line client on windows

## how to clear command lines

```sql
\! cls
```

## list all available databases

```sql
show databases;
```

## create new database

```sql
CREATE DATABASE <name>
CREATE DATABASE <snake_case>
CREATE DATABASE <camelCase>
```

## delete datatabse

```sql
 DROP DATABASE <name>
```

## use database (crud)

```sql
 USE <database>
```
